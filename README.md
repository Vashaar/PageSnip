# PageSnip

PageSnip is a simple, offline PDF splitter that lets you extract pages and compress files without uploading anything to third-party websites.

No subscriptions. No file limits. Your files stay on your computer.

---

## What it does

- Split PDFs using custom page ranges like `1-3, 6, 9-12`
- Drag and drop files or choose them manually
- Export only the pages you need
- Optional compression for smaller file sizes
- Works completely offline

## Why PageSnip?

Most online PDF tools:
- require you to upload private files  
- limit features behind paywalls  
- reduce quality during compression  

PageSnip keeps everything local, fast, and simple.

---

## How to Use

1. Open PageSnip  
2. Drag and drop your PDF (or select it manually)  
3. Enter page ranges (example: `1-3, 6, 9-12`)  
4. Choose compression (optional)  
5. Click export  

---

## Compression Modes

- **Original Quality**  
  Preserves full PDF quality and text selectability  

- **Balanced / Compact**  
  Reduces file size by converting pages to images  
  (smaller size, but text may not be selectable)
