# PageSnip v1.0.0

A lightweight desktop PDF splitter for Windows with a modern GUI.

## Included in this release

- PDF import via file picker, drag and drop, and clipboard path
- Flexible page range selection
- Split PDF export
- Optional compression modes
- Friendly activity panel with useful status updates
- Updated custom app icon and branding

## Installer

- `PageSnip-1.0.0-win-x64.exe`

## Notes

- `Original quality` preserves normal PDF fidelity and text selectability
- `Balanced` and `Compact` reduce file size more aggressively, but text may no longer be selectable
